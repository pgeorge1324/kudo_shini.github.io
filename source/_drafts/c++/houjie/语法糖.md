# auto 

```c++
list<string> c;
list<string>::iterator ite;
ite = find(c.begin(),c.end(),target);
// ==>
auto ite = find(c.begin(),c.end(),target);
```

# ranged-base for

```c++
for(decl:collection){
	statement
}
// 例
vector<double> vec;
for(auto elem:vec){
	cout << elem << endl;
}
for(auto& elem:vec){
	cout << elem << endl;
}
// 注意传引用和传值的区别
<==> iterator foreach
```
